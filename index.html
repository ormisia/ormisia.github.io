🛠️ 环境准备 & 必备工具

✅ 推荐唯一 IDE：Visual Studio Code（VSCode）

CTF 大多数题目不需要大量写代码，但写脚本、调试程序、逆向分析等操作仍需基本工具。我们推荐统一使用 VSCode：

  • 下载地址：https://code.visualstudio.com/

  • 推荐插件（安装后自动提示）：

    • Python（用于写脚本）
    • C/C++（便于调试逆向）
    • Hex Editor（用于查看二进制）
    • CodeLLDB（用于Pwn调试）
    • REST Client / Thunder Client（用于 Web 接口测试）
    • 010 editor（16进制编辑器）

  • Python 安装：https://www.python.org/（推荐Python 3.10+，）

  • 其他必备工具：

    • DBG / IDA Free（逆向分析）
    • pwntools（Pwn神器）
    • Burp Suite（Web抓包神器）
    • CyberChef（在线万能数据处理器）
    • binwalk / steghide / zsteg（图像隐写）
    • antsword
    • Typora（markdown文件处理轻量化的笔记软件）
    • VMware（在一个软件里面就能体验操控多个虚拟机）

🧠 常见问题 Q&A

Q：完全不会也能参加？ A：可以！我们培训从零开始，萌新赛也是“专为小白设计”的友好题。只要你有好奇心、有冲劲，我们都欢迎。

Q：CTF是不是很难？ A：一开始的确有点懵，但我们有完善的培训路径、学长手把手带，只要你坚持做题，很快会有质变。

Q：我不是计算机类专业，可以参加吗？ A：当然可以！我们有来自电信、机械、数院、经管、英语的选手，CTF是开放且多元的。

Q：加入之后能得到什么？ A：

  • 技术飞跃，熟练掌握Linux、调试、脚本、逆向分析等
  • 高质量项目参与+实战经历，助力简历、比赛和保研
  • 战队荣誉、国省奖项、与企业大厂接轨的真实体验
  • 社团影响力+“大神”光环，你会发现“你就是传说”

✅ 如何准备？

  • 提前学点 Python（B站搜“Python入门”，“Jack老师的脚本课”）
  • 注册 CTFHub、BUUCTF 尝试简单题
  • 准备一台电脑，建议安装 WSL2 或虚拟机，开始熟悉 Linux 命令
